# README

```bash
    chezmoi init git@github.com:chberreth/dotfiles.git
```

Current state requires to install `unzip`, `pipx` to use the dofiles. Afterwards, just run

```bash
    chezmoi init git@github.com:chberreth/dotfiles.git

    # eventually
    chmod +x ~/bin/chezmoi

    chezmoi diff

    chezmoi apply
```
